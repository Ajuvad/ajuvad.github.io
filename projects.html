AI-Based Assistance System for Visually Impaired

Developed an AI-powered system to assist visually impaired individuals in navigating their surroundings safely. The system uses real-time camera input and deep learning to detect obstacles and understand the environment, then provides instant audio feedback to guide the user. Built using Python, OpenCV, and YOLO, the model was trained on a custom dataset (Roboflow) to improve detection accuracy. The solution offers a hands-free, user-friendly approach to enhance independence and safety in both indoor and outdoor environments.

Synapse Protocol – Adaptive AI Combat System

Developed an adaptive AI combat system that replaces traditional scripted game AI with intelligent agents capable of learning from player behavior. The system uses Multi-Agent Reinforcement Learning (MAPPO) to enable real-time adaptation, allowing enemies to adjust their strategies based on how the player plays. GOAP (Goal-Oriented Action Planning) is used for high-level strategies like flanking and ambush, while Graph Neural Networks (GNN) enable coordination between multiple AI agents for team-based actions. A player behavior analysis module further enhances adaptability. This project demonstrates the ability to design and integrate learning, planning, and multi-agent communication into a unified AI system, resulting in more realistic, dynamic, and engaging gameplay.
